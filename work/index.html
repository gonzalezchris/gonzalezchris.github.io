My Work

I work at Told Media as a React Native Developer. Before this I was working for Mappify, a startup I co-founded. I've also worked on mobile applications for various startups, my latest work being Pet's I've Met. Pet's I've Met was developed with React Native and Redux.

Although my primary focus is mobile development, I've also done some frontend and backend development for a number of web applications. These include the Mappify platform, and other smaller projects such as Stories Of The Stolen, an online memorial for those who's lives have been taken by terrorist groups. The frontend for these projects involved the use of JQuery and React, while the backend utilized Node, Mongoose, and Express.
